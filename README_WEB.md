# Веб-интерфейс для Королевской биржи

## Запуск

```bash
python3 run_web.py
```

Или напрямую через uvicorn:
```bash
uvicorn web_server:app --host 0.0.0.0 --port 8000
```

## Доступ

Откройте в браузере: `http://localhost:8000`

Для доступа с других устройств в той же сети: `http://[IP_АДРЕС_КОМПЬЮТЕРА]:8000`

## Функции

- **Турнирная таблица**: место, имя игрока, капитализация, прирост от предыдущего раунда
- **Цены на ресурсы**: текущая цена, изменение от предыдущего раунда, изменение с начала игры
- **Построенные объекты**: количество каждого типа объектов и процент игроков, владеющих ими

## Обновления

Интерфейс обновляется автоматически через WebSocket каждую секунду.

## Интеграция с игрой

Для использования с реальной игрой нужно передать экземпляр игры в веб-сервер:

```python
from web_server import set_game
from game_engine import Game

game = Game(num_players=30)
set_game(game)
```

Затем запустить сервер.
